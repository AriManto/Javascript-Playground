Explanation

Sometimes the URL characters need to be encoded in ASCII format to prevent the program from interpreting it as an instruction or cutting the URL. For example, a space in the URL is indicated with a '%20', '+' as '%2B', '@' as '%40', etc.

This tool allows you to translate the ASCII code in an URL to the respective value. It is useful for normalizing the URL.

Example

  • ASCII Encoded URL:

    https%3A%2F%2Fdeveloper.mozilla.org%2Fen-US%2Fdocs%2FWeb%2FJavaScript%2FGuide%2FControl%5Fflow%5Fand%5Ferror%5Fhandling

  • Normalized URL:

    https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Control_flow_and_error_handling
